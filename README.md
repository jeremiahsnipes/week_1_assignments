# ANA 680 - Module 1 Assignments

This repository contains all three assignments for Module 1 of ANA 680: Machine Learning Deployment.

## Contents
- **Activity 1 – Implementing ML Lifecycles using Python**  
  Jupyter notebook predicting housing prices (Ames dataset) using Linear Regression, Decision Tree, Random Forest, and Gradient Boosting.

- **Assignment 2A – Building Classification Models**  
  Notebook comparing traditional classifiers (Logistic Regression, Decision Tree, Random Forest, KNN, SVM) on a labeled dataset.

- **Assignment 2B – Designing a Three-Layer ANN Classifier**  
  Notebook implementing a simple neural-network classifier in Keras/TensorFlow.

All code developed in Python with scikit-learn, pandas, and TensorFlow within a Jupyter/Anaconda environment.
